# Fase 0.5: Micro-Decisoes (10 min)

> **Esta e a fase mais importante.** Cada decisao nao tomada aqui = retrabalho caro depois.

## Quem faz: Humano

## Template: [`../../templates/decisoes.md`](../../templates/decisoes.md)

## 10 Categorias de Decisao

| Cat | O que define | Por que evita retrabalho |
|-----|-------------|--------------------------|
| A | Responsivo (3 frames: 375/768/1440) | "Nao cabe no mobile" |
| B | Conteudo (tipo de pagina, ordem, CTAs) | Reorganizar depois de pronto |
| C | Visual (cores, fontes, type scale) | "Ajusta essa fonte" 15x |
| D | Imagens (formato, budget, crops) | Hero de 2MB matando PageSpeed |
| E | Formulario (campos, validacao, pos-submit) | Recodar o form 3x |
| F | Performance (budgets, loading, animacoes) | PageSpeed 40 no final |
| G | SEO (URL, indexacao, schema, OG) | Refazer meta tags |
| H | Interacoes (nav, sticky, hover, feedback) | "Cade o botao no mobile?" |
| I | Tracking (eventos GTM, UTMs, thank you) | "Conversao nao rastreia" |
| J | Entrega (pastas, git, versionamento) | Bagunca com N clientes |
