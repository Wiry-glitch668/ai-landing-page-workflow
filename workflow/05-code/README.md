# Fase 4: Figma para Codigo (20 min)

## Quem faz: IA + Humano

## Prompt: [`../../prompts/figma-to-code.md`](../../prompts/figma-to-code.md)

## Processo

1. Anexar `decisoes.md` junto com o prompt
2. IA le o Figma (via MCP ou screenshot) + decisoes
3. IA gera projeto React/Tailwind/Vite completo
4. Humano revisa em 5 viewports (320, 375, 390, 768, 1440)
5. Humano roda Lighthouse e verifica > 90

## Stack padrao
- React 18+ TypeScript strict
- TailwindCSS
- Vite
- Sem dependencias desnecessarias
