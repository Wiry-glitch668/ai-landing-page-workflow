# Fase 2: Wireframe no Figma (10 min)

## Quem faz: Humano

## Processo

1. Abrir biblioteca de referencias no Figma
2. Filtrar por tipo (lead capture, vendas, VSL, agendamento)
3. Verificar se numero de secoes bate com a copy
4. Duplicar para novo arquivo: `[Cliente] - Landing Page v1`
5. **Criar 3 frames:** Mobile 375px | Tablet 768px | Desktop 1440px
6. Reorganizar secoes conforme estrutura da copy

## Criterios de Selecao

- Tipo bate com o objetivo?
- Secoes suficientes para a copy?
- Tem espaco para os elementos? (video, form, mapa)
- Estilo combina com o segmento?
