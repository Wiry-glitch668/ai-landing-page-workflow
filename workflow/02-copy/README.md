# Fase 1: Copy de Vendas (15 min)

## Quem faz: IA + Humano

## Prompt: [`../../prompts/copy-vendas.md`](../../prompts/copy-vendas.md)

## Processo

1. Colar o brief completo no prompt
2. IA gera 2 versoes (emocional + direta)
3. Humano escolhe, mescla, e refina a voz

## Sequencia Psicologica de Conversao

A copy segue esta ordem (cada secao responde uma pergunta do visitante):

1. "Isso e pra mim?" → Hero + Problema
2. "Vale a pena?" → Solucao + Beneficios
3. "Funciona mesmo?" → Prova Social
4. "Posso confiar?" → Credenciais
5. "E se der errado?" → FAQ + Garantia
