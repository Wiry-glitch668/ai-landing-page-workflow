# Checklist Pre-Launch

> Verificar TODOS os itens antes de colocar no ar.

## Conteudo
- [ ] Todo placeholder removido
- [ ] Headline comunica valor em < 3 segundos
- [ ] Sem links externos que vazam visitantes
- [ ] Navegacao removida ou minima

## Tecnico
- [ ] SSL ativo (HTTPS)
- [ ] Favicon presente
- [ ] OG tags configuradas
- [ ] Canonical URL configurada
- [ ] robots.txt ok
- [ ] sitemap.xml gerado

## Performance (Core Web Vitals)
- [ ] LCP ≤ 2.5s
- [ ] INP < 200ms
- [ ] CLS < 0.1
- [ ] PageSpeed mobile ≥ 90
- [ ] Peso total < 1.5MB
- [ ] Imagens WebP/AVIF comprimidas
- [ ] CSS/JS minificados
- [ ] Gzip/Brotli habilitado

## Responsivo
- [ ] 375px (iPhone SE) — ok
- [ ] 390px (iPhone 14) — ok
- [ ] 768px (iPad) — ok
- [ ] 1440px (Desktop) — ok
- [ ] 320px (stress test) — nada quebra

## Formulario
- [ ] Submissao funciona
- [ ] Dados chegam no destino
- [ ] /obrigado carrega
- [ ] Erro claro se falhar

## Analytics
- [ ] GTM instalado sem erros
- [ ] GA4 pageview funcionando
- [ ] Conversao = Key Event
- [ ] Ads conversion tag (se aplicavel)
- [ ] Pixel Meta (se aplicavel)
- [ ] Eventos disparando
- [ ] UTMs capturados
- [ ] Lead teste submetido e verificado

## LGPD
- [ ] Cookie consent antes de cookies
- [ ] Banner PT-BR
- [ ] Aceitar/Recusar iguais
- [ ] Cookies bloqueados ate aceite
- [ ] Privacidade linkada
- [ ] Opt-in marketing explicito

## Acessibilidade (WCAG 2.1 AA)
- [ ] Contraste ≥ 4.5:1
- [ ] Alt text em imagens
- [ ] Keyboard nav funciona
- [ ] Focus indicators visiveis
- [ ] Labels nos inputs
- [ ] Font ≥ 16px body

## Final
- [ ] Testado em janela anonima
- [ ] Testado em celular real
- [ ] Lead teste completo
- [ ] GA4 DebugView ok
- [ ] Backup/screenshot feito
