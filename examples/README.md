# Exemplos

Esta pasta contem exemplos reais (anonimizados) de briefs e decisoes preenchidos.

## Como Contribuir com Exemplos

1. Copie os templates de `../templates/`
2. Preencha com dados reais do seu projeto (anonimize dados sensiveis)
3. Adicione um `resultados.md` com metricas obtidas
4. Abra um PR ou Issue com a tag `example`

## Estrutura de um Exemplo

```
examples/
  clinica-odontologica/
    brief.md
    decisoes.md
    resultados.md
  ecommerce-moda/
    brief.md
    decisoes.md
    resultados.md
```

> Nunca inclua dados reais de clientes, tokens de API, ou informacoes sensiveis.
